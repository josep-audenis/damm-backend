# Decision Log

Use this folder for contract-impacting decisions.

Each decision should include:

- Date.
- Status: proposed, accepted, rejected, superseded.
- Context.
- Decision.
- Frontend impact.
- Backend impact.
- Migration notes.

Template:

```md
# YYYY-MM-DD Short Decision Title

**Status:** proposed

## Context

## Decision

## Frontend Impact

## Backend Impact

## Migration Notes
```
